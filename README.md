### 📌 Programação Python e Banco de Dados MySQL

**Descrição Geral**
Curso voltado ao aprendizado de desenvolvimento de software utilizando **Python** integrado a **banco de dados MySQL**, com abordagem didática e orientada a iniciantes na área de tecnologia.

### 🔹 Tecnologias Utilizadas

* **Python** – Linguagem de programação para desenvolvimento de aplicações.
* **MySQL** – Sistema de gerenciamento de banco de dados relacional.
* **SQL** – Linguagem padrão para manipulação e consulta de dados.
* **Interface Gráfica (GUI)** – Desenvolvimento de telas para aplicações desktop.

### 🔹 Principais Conceitos Abordados

* Fundamentos da **linguagem Python** para desenvolvimento de software.
* Introdução ao **MySQL** e criação de bancos de dados.
* Estrutura e uso da **linguagem SQL** para manipulação de dados.
* Implementação do padrão **CRUD**:

  * **Create** – Criação de registros
  * **Read** – Consulta de dados
  * **Update** – Atualização de registros
  * **Delete** – Remoção de dados

### 🔹 Projeto Prático

* Desenvolvimento de **um software completo em Python**.
* Integração entre **interface gráfica, lógica de programação e banco de dados MySQL**.
* Implementação de operações **CRUD** conectadas ao banco de dados.
* Construção de um projeto que pode ser utilizado como **portfólio profissional**.

---

✅ **Competências Desenvolvidas**

* Programação em **Python para iniciantes**
* Modelagem e manipulação de **bancos de dados MySQL**
* Uso de **SQL para operações em banco de dados**
* Desenvolvimento de **sistemas com CRUD**
* Integração entre **aplicação Python, interface gráfica e banco de dados**
